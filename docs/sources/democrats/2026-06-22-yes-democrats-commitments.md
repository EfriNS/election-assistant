# הדמוקרטים — התחייבויות מדיניות (yes.democrats.org.il)

**מקור:** https://yes.democrats.org.il  
**תאריך איסוף:** 2026-06-22  
**הערה:** דף ההתחייבויות של הדמוקרטים לקראת הבחירות 2026. הנתונים נשלפו מהעמודים הנושאיים. עמוד "דמוקרטיה וצדק" לא נטען (JavaScript-only rendering) — יש לאסוף ידנית.

## ביטחון ודיפלומטיה
מקור: https://yes.democrats.org.il/topics/security-diplomatic

- נוביל יוזמה מדינית אזורית ונקדם נורמליזציה עם סעודיה
- נביא לסגירת כל זירות הלחימה - עזה, לבנון וסוריה
- נבלום את הסיפוח, נילחם בטרור היהודי ונקדם אלטרנטיבה פלסטינית מתונה
- נילחם ברדיקלים ונחזק את המתונים - נחליף את חמאס בעזה, נילחם ביד קשה בטרור
- נקים ועדת חקירה ממלכתית ונשקם את מערכת הביטחון, ננקה אותה מפוליטיקה

## כלכלה וחברה
מקור: https://yes.democrats.org.il/topics/economy-society

- במקום כסף למקורבים ולמשתמטים, נגדיל משמעותית את ההשקעה בשירותים חברתיים
- נדאג לצרכי המשפחות הצעירות, נשפר את תנאי העבודה, נשפר את איכות המסגרות
- מלחמת חורמה ביוקר המחיה באמצעות מלחמה במונופולים
- נדאג לדיור בר השגה - התחדשות עירונית, שכירות ארוכת טווח ודיור ציבורי
- נוביל מהלך לאומי לשיקום העוטף וקו העימות ולשגשוג הצפון והדרום

---

## חינוך
מקור: https://yes.democrats.org.il/topics/education

- נבטל את האפליה התקציבית נגד החינוך הממלכתי, ונוביל לשוויון בין כל תלמידי ישראל
- נקים את המועצה לחינוך ממלכתי. נייצר לו עמוד שדרה, יציבות ושומרי סף.
- נחייב חינוך לאזרחות ודמוקרטיה לכולם.
- נחזק את מעמד עובדי ועובדות ההוראה והניהול ונשפר את תנאי ההעסקה.
- נפסיק את מימון הרשתות החרדיות המפלגתיות - חינוך ממלכתי-חרדי לכל ילד חרדי.
- נשקם את מערך החינוך לגיל הרך.

## דת ומדינה
מקור: https://yes.democrats.org.il/topics/religion-state

- נחוקק חוק נישואין אזרחיים ונאפשר חופש בחירה בנישואים וגירושים
- נפרק את מונופול הרבנות בגיור ובכשרות
- נאבק על מרחב ציבורי חופשי ונקדם תחבורה ציבורית בשבת
- נילחם בהדתה ובאפליה בצה"ל, במרחב הציבורי ובמערכת החינוך
- נוביל שיוויון מוחלט בנטל

---

## שוויון (קהילת הלהט"ב)
מקור: https://yes.democrats.org.il/topics/lgbtq-equality

- נקדם שוויון זכויות מלא בחוק לקהילה הגאה, כולל שוויון בזכות להורות, למשפחה ולנישואים
- נאסור טיפולי המרה, נחזק את האכיפה נגד אפליה, נכיר בנפגעי פשעי שנאה ונפסיק את התמיכה הציבורית בארגונים להט"בופוביים
- נטמיע במערכת החינוך תכנים המעודדים כבוד, סובלנות, קבלה ושוויון לכל אדם
- נגדיל את תקציבי התמיכה, נחזק את הקהילה הגאה למען חברה פתוחה, מגוונת וסובלנית יותר
- נשפר את הטיפול המשטרתי בפשעי שנאה, נחזק את ההגנה על אוכלוסיות בסיכון

## דמוקרטיה וצדק
מקור: https://yes.democrats.org.il/topics/democracy-justice  
**הערה:** עמוד מוגש ב-JavaScript בלבד; הטקסט נאסף ידנית ע"י המשתמש (2026-06-22).

- נבצר את הזכויות האזרחיות והחברתיות בחוקי יסוד וניתן להם מעמד עליון
- נחזק את הכנסת ואת עקרון הפרדת הרשויות
- נבטיח כי בכירים בצמרת הביטחון ובשירות המדינה ועובדי ציבור ישרתו את החוק
- נבטל את כל חוקי ההפיכה המשטרית, ונחוקק את חוק יסוד החקיקה
- נחוקק את חוקי הגבלת הכהונה ונלחם בשחיתות
